Modification des propriétés de plusieurs comptes d'utilisateurs simultanément

Cette démonstration explique comment modifier simultanément des propriétés de plusieurs comptes d'utilisateurs à l'aide du composant Utilisateurs et ordinateurs Active Directory.

  1. Sélectionnez les comptes d'utilisateurs dont vous souhaitez modifier les propriétés, cliquez avec le bouton droit sur la sélection, puis cliquez sur Propriétés.
  2. Dans la boîte de dialogue Propriétés d'objets multiples, effectuez l'une des tâches suivantes :
    • Si la propriété requiert une description texte, activez la case à cocher appropriée pour activer la zone de texte, puis tapez la modification.

    • Si la propriété doit être sélectionnée, activez la case à cocher appropriée, puis sélectionnez la modification.
  3. Cliquez sur Appliquer pour appliquer la modification.
