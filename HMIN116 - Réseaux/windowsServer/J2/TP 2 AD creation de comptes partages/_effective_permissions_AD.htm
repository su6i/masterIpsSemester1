Comment vérifier les autorisations effectives pour gérer un objet Active Directory

Définition des autorisations effectives

Les autorisations effectives pour un objet du service d'annuaire Active Directory regroupent les autorisations Active Directory accordées aux comptes d'utilisateurs et aux comptes de groupes.
Vous pouvez utiliser l'outil Autorisations effectives pour déterminer les autorisations pour un objet Active Directory. Cet outil calcule les autorisations accordées à un utilisateur ou à un groupe spécifique et tient compte des autorisations résultant de l'adhésion à un groupe ainsi que des autorisations héritées des objets parents.
Pour récupérer les informations relatives aux autorisations effectives dans Active Directory, vous devez être autorisé à lire les informations d'appartenance.
La démonstration suivante montre comment vérifier les autorisations effectives accordées pour gérer un objet Active Directory.

Remarque : cette démonstration utilise l'option Fonctionnalités avancées de Utilisateurs et ordinateurs Active Directory. Pour activer cette option, dans le menu Affichage de Utilisateurs et ordinateurs Active Directory, cliquez sur Fonctionnalités avancées.

  1. Dans Utilisateurs et ordinateurs Active Directory, cliquez avec le bouton droit sur une unité d'organisation ou un objet compte, puis cliquez sur Propriétés.
  2. Sous l'onglet Sécurité de la boîte de dialogue Propriétés, cliquez sur Avancé.
  3. Sous l'onglet Autorisations effectives de la boîte de dialogue Paramètres de sécurité avancés, cliquez sur Sélectionner.
  4. Dans la zone Entrez le nom de l'objet à sélectionner de la boîte de dialogue Sélectionnez Utilisateur, Ordinateur ou Groupe, tapez le nom d'un utilisateur ou d'un groupe, puis cliquez sur OK.

Les cases à cocher activées indiquent les autorisations effectives sur l'objet accordées à l'utilisateur ou au groupe sélectionné.
