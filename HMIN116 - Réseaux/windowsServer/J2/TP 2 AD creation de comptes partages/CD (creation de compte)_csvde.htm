Utilisation de CSVDE pour créer des comptes d'utilisateurs

CSVDE (Comma-Separated Value Directory Exchange) permet d'ajouter de manière efficace un grand nombre d'utilisateurs au service d'annuaire Active Directory. Les informations de la plupart des bases de données peuvent être exportées au format universel CSV. Après modification, ces informations peuvent être importées directement dans Active Directory. Vous ne pouvez pas utiliser le format CSVDE pour supprimer ou modifier des objets dans Active Directory. Pour garantir le succès de l'importation, vous devez vérifier que le fichier est bien au format CSVDE. En règle générale, pour modifier et mettre en forme un fichier texte, vous devez utiliser une application offrant des fonctionnalités d'édition évoluées, telle que Microsoft Excel. Il vous suffit ensuite de l'enregistrer en tant que fichier texte délimité par des virgules (CSV).

Lorsque vous créez plusieurs comptes d'utilisateurs, les informations contenues dans le fichier CSV doivent comprendre le chemin d'accès du compte d'utilisateur dans Active Directory, le type d'objet (le compte d'utilisateur), ainsi que le nom d'ouverture de session de l'utilisateur. En revanche, elles ne peuvent pas comprendre de mots de passe. Les mots de passe seront tous vides et devront être modifiés par les utilisateurs.

Mettez en forme le fichier CSV de sorte que la première ligne soit une ligne d'attributs et que chaque compte d'utilisateur possède une ligne supplémentaire. La ligne d'attributs spécifie le nom de chaque attribut à définir pour l'ensemble des nouveaux comptes d'utilisateurs. Chaque ligne supplémentaire spécifie les valeurs de chaque compte d'utilisateur. S'il manque une valeur d'attribut, laissez-la vide mais insérez toutes les virgules nécessaires. Si une valeur comporte des virgules, entourez-la de guillemets. L'exemple suivant comprend deux des comptes d'utilisateurs que vous devez créer :

DN,objectClass,cn,sn,givenName,displayName,name,countryCode, userPrincipalName,objectCategory

"CN=John Smith,ou=CSO,DC=contoso,DC=com",user,John Smith,Smith,John,John Smith,John Smith,0,smithj@contoso.com,"CN=Personne,CN=Schéma,CN=Configuration, DC=contoso,DC=com"

"CN=Kelly Blue,ou=CSO,DC=contoso,DC=com",user,Kelly Blue,Blue, Kelly,KellyBlue,KellyBlue,0,bluek@contoso.com,"CN=Personne,CN=Schéma, CN=Configuration,DC=contoso,DC=com"

Une fois le fichier CSV créé, vous pouvez l'importer en exécutant la ligne de commande suivante à partir de l'invite de commandes :

CSVDE –i –f fichier.csv
