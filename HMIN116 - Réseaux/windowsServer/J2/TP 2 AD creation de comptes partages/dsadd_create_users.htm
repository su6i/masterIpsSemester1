Utilisation de Dsadd pour créer des comptes d'utilisateurs

Dsadd est un outil de ligne de commande qui permet d'ajouter un utilisateur ou un autre objet au service d'annuaire Active Directory. Dsadd n'ajoute qu'un seul utilisateur à la fois. Toutefois, il s'utilise généralement dans des fichiers de commandes pour automatiser l'ajout d'objets dans Active Directory. De manière générale, l'utilisation du composant Utilisateurs et ordinateurs Active Directory pour créer moins d'une douzaine de comptes d'utilisateurs est plus rapide et génère moins d'erreurs.

Pour créer un compte d'utilisateur à l'aide de dsadd user, tapez ce qui suit à l'invite de commandes ou dans un fichier de commandes :

dsadd user DN_Utilisateur [-samid Nom_SAM]
[-upn UPN] [-fn Prénom] [-ln Nom] [-display Nom_Affiché]
[-pwd {Mot_passe|*}] [-disabled {yes | no}]

Pour afficher la syntaxe complète de cette commande, tapez dsadd user /? à l'invite de commandes. Utilisez la syntaxe " " pour les variables comportant des espaces.

L'exemple suivant illustre l'ajout d'un compte d'utilisateur nommé testuser au domaine contoso.msft :

dsadd user "cn=testuser,cn=users,dc=contoso,dc=msft" –samid testuser –upn testuser@contoso.msft –fn test –ln user –display "test user" –pwd P@ssw0rd – disabled yes

Les exemples suivants représentent quatre comptes d'utilisateurs que vous devez créer. Si vous les utilisez dans un fichier de commandes, remplacez le nom de domaine par le vôtre.


dsadd user "cn=Ido Ben-Sachar,ou=CSO,dc=contoso,dc=msft" -samid bensachari -upn bensachari@contoso.msft -fn Ido -ln Ben-Sachar -display "Ido Ben-Sachar" -pwd P@ssw0rd –disabled yes

dsadd user "cn=Jay Fluegel,ou=CSO,dc=contoso,dc=msft" -samid fluegelj -upn fluegelj@contoso.msft -fn Jay -ln Fluegel -display "Jay Fluegel" -pwd P@ssw0rd –disabled yes

dsadd user "cn=Jon Grande,ou=CSO,dc=contoso,dc=msft" -samid grandej -upn grandej@contoso.msft -fn Jon -ln Grande -display "Jon Grande" -pwd P@ssw0rd – disabled yes

dsadd user "cn=Alex Hankin,ou=CSO,dc=contoso,dc=msft" -samid hankina -upn hankina@contoso.msft -fn Alex -ln Hankin -display "Alex Hankin" -pwd P@ssw0rd – disabled yes
