Utilisation de Dsmod pour modifier des comptes d'utilisateurs

Vous pouvez utiliser la commande dsmod pour modifier les attributs d'un ou plusieurs utilisateurs dans le service d'annuaire Active Directory. Pour modifier les attributs d'un compte d'utilisateur :

Pour modifier les attributs d'un compte d'utilisateur à l'aide de dsmod user, tapez ce qui suit à l'invite de commandes ou dans un fichier de commandes :

dsmod user DN_Utilisateur ... [-upn UPN] [-fn Prénom] [-mi Initiales] [-ln Nom] [-display Nom_Affiché] [-pwd (Mot_passe | *)] [-tel Numéro_Téléphone] [-email Adresse_messagerie] [-fax Numéro_Télécopie] [-disabled {yes | no}]

Pour afficher la syntaxe complète de cette commande, tapez dsmod user /? à l'invite de commandes. Utilisez la syntaxe " " pour les variables comportant des espaces.

L'exemple suivant permet de modifier le numéro de télécopie de trois utilisateurs de l'organisation CSO (CSO, Customer Services Organisation) :

dsmod user "cn=Jay Fluegel,ou=cso,dc=contoso,dc=msft" "cn=Jon Grande,ou=cso,dc=contoso,dc=msft" "cn=Alex Hankin,ou=cso,dc=contoso,dc=msft" -fax "555-555-1212"
