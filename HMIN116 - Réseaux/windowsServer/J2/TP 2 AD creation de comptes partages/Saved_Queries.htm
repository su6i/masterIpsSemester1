Création d'une requête sauvegardée permettant de rechercher les comptes verrouillés

Dans ces travaux dirigés, vous allez créer, puis utiliser une requête sauvegardée pour rechercher un compte d'utilisateur verrouillé.

Tâches
Points clés
1. Ouvrez l'outil Utilisateurs et ordinateurs Active Directory. Dans Saved Queries, créez un dossier nommé Problem Accounts.     Créez des requêtes sauvegardées pour gagner du temps lors de la réalisation de tâches courantes, telles que la recherche d'un compte verrouillé.
2. Dans le dossier Problem Accounts, créez une nouvelle requête sauvegardée nommée Locked Accounts.     Les outils Requête sauvegardée et Rechercher vous permettent de créer des requêtes LDAP (Lightweight Directory Access Protocol) pour rechercher des objets Active Directory.
3. Cliquez sur Définir la requête, puis spécifiez une requête Recherche personnalisée.     Contrairement à l'interface utilisateur standard, le type de requête Recherche personnalisée vous permet de créer des requêtes LDAP, notamment pour les comptes verrouillés.
4. Dans la boîte de dialogue Rechercher recherche personnalisée, utilisez l'onglet Avancé pour spécifier une requête LDAP.

• Pour rechercher des comptes verrouillés, tapez la requête suivante sous forme de chaîne unique : “(&(&(&(objectCategory=personne)
(objectClass=utilisateur)
(lockoutTime:1.2.840.113556.1.4.804:
=4294967295))))”.

• Fermez la requête
  • L'onglet Recherche personnalisée vous permet de créer rapidement une requête utilisant de nombreux attributs, même si vous ne connaissez pas le langage de requête LDAP.
  • Utilisez l'onglet Avancé pour créer vos propres requêtes si les langages de requête LDAP vous sont familiers.
  • Entrez la chaîne de requête de recherche sans insérer de sauts de ligne
  • Si la requête n'est pas entrée correctement, vous serez informé que la chaîne de requête n'est pas valide.
  • Une fois créée, la requête s'exécute automatiquement. Toutefois, les résultats ne sont pas automatiquement actualisés à la prochaine sélection de la requête. Pour actualiser les résultats de la requête, sélectionnez la requête, puis appuyez sur F5.
5. Exportez la définition de requête dans un fichier nommé Find Locked Users.xml, dans Mes documents. Importez la requête à la racine du dossier Saved Queries sous le nom Find Locked Users-new. Vérifiez ensuite que la requête que vous venez d'enregistrer retourne les mêmes résultats que la requête d'origine.
Vous pouvez envoyer les requêtes sauvegardées par message électronique aux autres administrateurs.
